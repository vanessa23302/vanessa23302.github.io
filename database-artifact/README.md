Database Artifact – Travlr Getaways

This artifact demonstrates database interaction using MongoDB and Mongoose in the Travlr Getaways full-stack application developed in CS465.

The original code retrieved trip data from the MongoDB database using a basic query. The enhancement improves the functionality by allowing filtered database queries through API parameters. This allows the API to return specific records based on conditions instead of retrieving all trips.

This enhancement demonstrates improved database querying, backend API development, and integration between Express and MongoDB.
