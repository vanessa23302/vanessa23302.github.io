# Software Design and Engineering Artifact

## Overview
This artifact is the Contact Service application originally developed in CS 320: Software Testing, Automation, and Quality Assurance. The project demonstrates object-oriented programming in Java and includes classes that manage and validate contact information.

## Artifact Contents

### Original Code
The original implementation of the Contact Service application before enhancements.

### Enhanced Code
The enhanced version improves validation logic and ensures that contact information meets specific constraints such as maximum field lengths and non-null values.

### Narrative
The narrative explains the purpose of the artifact, the enhancements made, and the skills demonstrated in software design and engineering.

- Unit testing with JUnit
